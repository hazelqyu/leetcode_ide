# LeetCode IDE Workflow (VS Code)

This repo gives you a simple local workflow like an IDE plugin:

1. Pull a LeetCode problem into a local folder.
2. Edit `solution.py` in VS Code.
3. Run test input against LeetCode's judge.
4. Submit directly from terminal/task runner.

## Files

- `lcide.py`: CLI helper (`pull`, `run`, `submit`)
- `.vscode/tasks.json`: VS Code tasks so you can run everything from Command Palette/Tasks
- `problems/`: local problem folders generated by `pull`

## Setup

### 1) Python

Use Python 3.10+ (standard library only, no pip packages required).

### 2) Optional auth (required for `run` and `submit`)

`pull` works without login.

For `run` and `submit`, set these values from your logged-in LeetCode browser cookies:

- `LEETCODE_SESSION`
- `LEETCODE_CSRFTOKEN`

Recommended (one-time) setup:

```bash
cp .env.example .env
# then edit .env with real cookie values
```

You can still use shell exports if you prefer. `.env` values are loaded automatically.

Optional config (defaults shown):

```bash
LEETCODE_BASE='https://leetcode.com'
LCIDE_WORKSPACE='problems'
```

## Usage

### Roadmap (NeetCode categories)

One-time sync (creates `data/neetcode150.json`):

```bash
python3 lcide.py roadmap sync
```

List categories:

```bash
python3 lcide.py roadmap categories
```

List problems in a category:

```bash
python3 lcide.py roadmap problems --category "Two Pointers"
```

Pull from roadmap by index in category:

```bash
python3 lcide.py roadmap pull --category "Two Pointers" --index 1
```

Or pull by problem title:

```bash
python3 lcide.py roadmap pull --title "3Sum"
```

### Pull a problem

```bash
python3 lcide.py pull two-sum
```

Creates:

- `problems/0001-two-sum/prompt.md`
- `problems/0001-two-sum/solution.py`
- `problems/0001-two-sum/example_testcases.txt`
- `problems/0001-two-sum/meta.json`

### Run your code (remote judge)

```bash
python3 lcide.py run two-sum
```

Or custom input:

```bash
python3 lcide.py run two-sum --input $'[-1,-2,-3,-4,-5]\n-8'
```

### Submit

```bash
python3 lcide.py submit two-sum
```

## VS Code Tasks

Use `Terminal -> Run Task`:

- `LeetCode: Pull Problem`
- `LeetCode: Run (Remote)`
- `LeetCode: Submit`

Each task prompts for slug (for example, `two-sum`).

## Notes

- This is Python-first (`solution.py`, `lang=python3`).
- `run` uses LeetCode's interpreted run endpoint (online judge), not a local simulator.
- Roadmap data is stored in `data/neetcode150.json` after sync.
- If LeetCode API behavior changes, endpoints/payloads may need updates.
